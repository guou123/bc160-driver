# bc160-driver
全网第一份独家的BC160驱动，带显示接口和没有显示接口的都能用
# BC-160 专用驱动 (基于 W5700X 魔改) 

## 适用显卡 
- AMD BC-160 (Navi 12, 8GB HBM2)
  
## 安装方法 
1. 解压驱动包
2. 设备管理器手动更新驱动，指向解压后的文件夹
3. 选择 W5700X 驱动
4. 具体看哔站的[](我还没录，详细教程等一段时间)

## 已测试 
- Deepseek 18B 50-60 tokens/s
- 鲁大师 3D 80-100fps
- 挖矿比 RTX 3080 强（手动狗头）

## 注意事项 
- 仅限 BC-160 使用，其他卡勿试
- 安装前请备份原驱动
- 有任何问题请在 Issues 反馈
